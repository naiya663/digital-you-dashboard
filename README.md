# 📊 Digital You Dashboard

A Python-based productivity tracking application that helps users analyze their study habits using data visualization and insights.

## 🚀 Features
- Track daily study hours
- Subject-wise analysis
- Pie chart & bar graph visualization
- Smart productivity insights
- User-friendly GUI using Tkinter

## 🛠 Tech Stack
- Python
- Pandas
- Matplotlib
- Tkinter
